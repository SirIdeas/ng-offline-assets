ng-offline-assets

Usage

Installation

Bower installation

$ bower install ng-offline-assets --save

Add javascript file

For example

<script src="ng-offline-assets/dist/js/ng-offline-assets.js"></script>

Angular module dependency

angular.module('app', ['ngOfflineAssets']);

Examples

Directive oa-bg

<div oa-bg="'/assets/sirideas.png'"></div>
// Local: {{localUrlBg}}

Directive oa-src

<img oa-src="'/assets/sirideas.png'">
// Local: {{localUrlSrc}}
